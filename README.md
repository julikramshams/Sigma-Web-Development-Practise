## 🎯 About This Repository

This repository is my personal code diary and portfolio for the **Sigma Web Development Course** by Haris Ali Khan (CodeWithHarry). 

Every folder and line of code in this repository represents active learning. Rather than just passively watching tutorials, I use this space to type out code, debug errors, experiment with layouts, and deeply solidify my understanding of web development fundamentals.

> **To Recruiters & Peers:** This repository demonstrates my consistency, my commitment to mastering the fundamentals before relying on heavy frameworks, and my ability to follow through on a massive, comprehensive curriculum.
